## Deployment

`composer update`<br>
copy `.env.example` kemudian rename menjadi `.env`<br>
atur database:<br>
`DB_DATABASE=app_iot
DB_USERNAME=root
DB_PASSWORD=`<br>

buat database bernama <b>app_iot</b><br>

`php artisan key:generate`<br>
`php artisan migrate --seed`<br>
`php artisan serve`<br>
jalankan http://127.0.0.1:8000/ di browser<br>
